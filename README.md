# ✅ My To-Do List App

A simple and efficient **To-Do List Web Application** built using **React.js**. This app allows users to manage their daily tasks with ease — add, delete, and search todos with real-time UI updates and persistent local storage.

## ✨ Features

- ➕ Add tasks with serial numbers, titles, and descriptions
- ❌ Delete tasks with a single click
- 🔍 Real-time search through title and description
- 💾 Persistent storage with `localStorage`
- 🔁 Automatic updates to UI on data change
- 🧭 Route navigation using React Router (`/` and `/about`)
- 📱 Fully responsive UI

---

## 🛠️ Tech Stack

- ⚛️ **React.js**
- 🧭 **React Router DOM**
- 💅 **Bootstrap 5**
- 🗂 **Local Storage** (for client-side persistence)
- 🎨 **Custom CSS**
